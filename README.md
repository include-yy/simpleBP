# simpleBP -- ReScript 实现的简单 BP 网络

## Installation

```sh
git clone https://github.com/include-yy/simpleBP
cd simpleBP
npm install
```

前往 [MNIST](http://yann.lecun.com/exdb/mnist/)，下载数据文件，解压后放入 `src` 目录中

## Build

- Build: `npm run res:build`

## Run

```sh
node Test.bs.js
node Mnist.bs.js
```

可参考 `Test.bs.js` 和 `Mnist.bs.js` 学习简单用法
